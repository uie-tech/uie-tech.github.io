## 关于本站

本站使用 [Hexo](https://hexo.io/) 搭建,采用 [Butterfly](https://github.com/jerryc127/hexo-theme-butterfly) 主题,部署在 GitHub Pages。

## 快速开始

```bash
# 新建文章
hexo new "我的新文章"

# 本地预览
hexo server

# 生成静态文件
hexo generate

# 部署到 GitHub
hexo deploy
```
